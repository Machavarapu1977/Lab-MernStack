# MERN Stack Lab Programs

This repository contains all the programs and experiments completed as part of the MERN Stack Lab. The aim of this lab is to understand and implement full-stack web development using MongoDB, Express.js, React, and Node.js.

## Project Overview

The MERN stack is a popular JavaScript-based framework used to build modern web applications. This repository demonstrates the integration of frontend and backend technologies to create dynamic and responsive applications.

## Technologies Used

- **MongoDB** – Database for storing application data  
- **Express.js** – Backend framework for handling server-side logic  
- **React.js** – Frontend library for building user interfaces  
- **Node.js** – Runtime environment for executing JavaScript on the server  

### Additional Tools
- Git & GitHub for version control  
- Postman for API testing  
- VS Code for development  

## Folder Structure


/Frontend -> React applications and UI components
/Backend -> Node.js and Express server code
/Database -> MongoDB connection and schema files
/Programs -> Individual lab exercises and implementations


## Features Covered

- Basic CRUD operations (Create, Read, Update, Delete)  
- REST API development  
- Frontend-backend integration  
- Form handling and validation  
- Database connectivity using MongoDB  
- Routing and state management in React  

## How to Run the Project

### 1. Clone the repository

git clone <your-repo-link>


### 2. Start Backend

cd backend
npm install
npm start


### 3. Start Frontend

cd frontend
npm install
npm start


### 4. Database Setup
Make sure MongoDB is running locally or use MongoDB Atlas.
